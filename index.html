Laronix Intelligibility Test

Thank you for participating Laronix's speech intelligibility test.

  • This test will last for 15 to 20 minutes.
  • Test contains 4 sets, please select 1 set only.

Instruction

  1. Input you name and select one set, then click start.
  2. Listen to the audio, and dictate what you heard.
  3. There are 4 sentences per page, after dictate ALL OF THEM you will see the next button, Click that to continue.
  4. If you can not tell what you heard, type / in the corresponding textbox. DON'T KEEP IT EMPTY.
  5. After finishing all the evaluation you will see the finish Button。Click it, and a file (your_name)_set_(n)_int.csv will be automatically downloaded.
  6. Send the csv file to

    7. admin@laronix.com

Miscellaneous

  • Please participate with PC rather than mobile phone.
  • Using Headphone or speaker under a quite environment.
  • Once the test starts, don't close or refresh the page until it's done.
  • Some sentences can not be played on the LAST PAGE, as long as you can see the finish button, it means that you have finished it correctly.

What is intelligibility (※Please read this carefully!)

  • The intelligibility is the degree to which speech sounds can be correctly identified and understood by listeners.
  • In this study, please ignore the influence from unsatisfactory naturalness or accent.
  • Try to dictate all the words you here even it can not form a sentence, but don't predict narrator's speech.
